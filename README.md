简介：
    这是一个离线的资源浏览器框架。现内容为多肉植物。
    使用了‘PyQt4’作为页面的基本框架。
    使用了爬虫从http://www.yuhuagu.com/抓取植物信息。
    后台数据库是使用python标准库sqilite3。
